# FlixMovies 🎬

A movie search web app built with React + Vite for my Front-End Developer Internship (Week 2).

🔗 **Live Demo:** https://movie-search-app-one-sandy.vercel.app/

## About

FlixMovies lets you search for any movie, view details like poster, title, year and rating, and save your favourites.
It uses the TMDB API for real-time data and runs entirely in the browser — no backend needed.

## Features

- Search movies by name
- Popular movies shown on load
- Skeleton loading UI while fetching
- Add or remove favourites with heart button
- Favourites saved even after page refresh
- Fully responsive on mobile, tablet and desktop

## Tech Used

- React + Vite
- React Router
- Axios
- Context API
- localStorage
- TMDB API
- CSS3
